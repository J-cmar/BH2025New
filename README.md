# 💊 Medifind
1st Place Winner & Best Web Dev – BroncoHacks 2025

Medifind is a comprehensive full-stack platform designed to make healthcare more accessible. It empowers users to find affordable medications, track safety via FDA recalls, and manage their health schedules through a unified, intuitive interface.

## 🚀 Features
Real-time Price Comparison: Scrapes current pricing data to help users find the most affordable options.

Drug Interaction Checker: Safety-first tool to analyze potential risks between different medications.

FDA Integration: Live updates on medication recalls and safety alerts directly from official FDA APIs.

Smart Reminders: Integrated calendar views and automated alerts to ensure users never miss a dose.

Pharmacy Locator: Interactive Google Maps integration to find the nearest provider.

Secure Profiles: Personalized medication storage and authentication powered by Supabase.

## 🛠️ Tech Stack
Frontend: Next.js (React), Tailwind CSS

Backend & Auth: Supabase (PostgreSQL)

Data Sourcing: Puppeteer (Web Scraping), FDA Open API

Maps & Scheduling: Google Maps Platform, FullCalendar / Custom Calendar UI

## 🏗️ Architecture
Medifind utilizes a modern serverless architecture to ensure speed and scalability:

Next.js handles the core application logic and UI rendering.

Puppeteer runs on the server side to fetch real-time pricing data that isn't available through standard APIs.

Supabase manages user sessions and stores sensitive medication schedules securely.
